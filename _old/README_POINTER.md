# Proyecto de vivienda original (archivo)

Esta carpeta contenía el **proyecto de asequibilidad de vivienda original**
(entregas 1–3 del curso), del que evolucionó el programa fiscal completo.

Para aligerar el repositorio, el contenido de `_old/` está **excluido del
control de versiones** (ver `.gitignore`); solo se conserva este puntero.

## Dónde encontrar el original

- **En el historial de Git**: el proyecto de vivienda completo está en el
  historial hasta el commit anterior a su exclusión, y de forma estable en la
  etiqueta **`entrega-4`** (`git checkout entrega-4`).
- **Localmente**: los ficheros siguen en `_old/` en la copia de trabajo del autor.
- **Su papel en el proyecto actual**: la transición de aquel proyecto a este,
  y cómo el análisis de vivienda se conserva como núcleo, está documentada en
  [`docs/entregas/anexo_transicion_proyecto.md`](../docs/entregas/anexo_transicion_proyecto.md).
